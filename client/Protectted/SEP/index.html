Class of September 2024 - Dashboard

Welcome to the class of Sept 2024 batch dashboard. On this page, you will find all the class related materials. We will be using this page to guide you all to follow up with the course.

Course materials

This section includes all the recorded live class sessions along with the practice exercises and weekly assignments you have to follow. All the pre-recorded lecture videos of the other batches can be found on the home page.

Phase 1: Building static websites using HTML & CSS

In this phase, you'll learn about the underlying structure of the web - HTML. You'll also learn how to apply styling to a website through CSS. To help you build websites faster and easier, you will learn the Bootstrap framework. Along the way, you'll also learn about VS Code editor, Chrome's Developer Tools, DNS configurations and deploying static websites.

Phase 2: Learn coding with JavaScript

JavaScript is among the most powerful and flexible programming languages of the web. It powers the dynamic behavior on most websites. In this phase of your course, you will learn programming fundamentals using the latest JavaScript syntax. The concepts covered in these lessons lay the foundation for using JavaScript in any environment.

Phase 3: Latest web development technologies (React, Node and Express)

React is a JavaScript library, developed in 2013 by Facebook. It is very popular and used on major sites including on Facebook, Netflix, Khan Academy and many others. In this phase, we will start from the fundamentals and go all the way up to building full applications with custom styling. Along the way, you'll also learn about npm and ES6.

Phase 4: The Project Phase - Building Fullstack Applications

In this crucial phase, we will consolidate your learning by diving into real-world projects, building functional clones of well-known web applications. This phase is designed to give you hands-on experience in fullstack development, integrating both front-end and back-end technologies.
